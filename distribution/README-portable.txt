Pro PDF Reader portable
=======================

Run ProPdfReader.exe to open the reader, or drag a PDF onto it.

To add this portable copy to Windows Open with and Default Apps, run:
  Register portable copy.cmd

If you move this folder, register it again from the new location. Before deleting
the folder permanently, run:
  Remove portable registration.cmd

PDF files and annotations are never copied into the application folder. Reading
state remains under %LocalAppData%\ProPdfReader for the current Windows account.
